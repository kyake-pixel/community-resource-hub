About This Website

This Community Resource Hub highlights local resources, events, businesses, and improvement opportunities. Developed by Keira Yake, it is designed to be user-friendly, interactive, and visually appealing. Below is the detailed work log and source documentation.

Project Overview

The Community Resource Hub is designed to centralize key information for residents, including:

  • Interactive Resource Directory with full details
  • Event Listings with modals for extra info
  • Community Highlights spotlighting key achievements
  • Shops & Food directory with location, hours, and contact info
  • Submit Improvement form for resident feedback
  • Animated backgrounds and elegant black/gold styling throughout
  • Responsive design for mobile and desktop

Challenges & Solutions

  • Animated backgrounds had slight flicker – refined CSS keyframes for smooth motion.
  • Card alignment inconsistent on mobile – used CSS grid auto-fit to ensure uniformity.
  • Modals sometimes overflowed – added max-height and scroll.
  • Footer placement caused empty space – adjusted section padding & min-height.
  • Event detail depth – expanded cards to include time, location, and description.

Notes & Future Considerations

  • Add search & filter functionality for resources.
  • Integrate real images (copyright-free) for resources & events.
  • Include Google Maps for location visualization.
  • Consider accessibility improvements for screen readers.
